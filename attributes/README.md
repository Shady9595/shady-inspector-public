# Attribute code samples

Place C# example scripts here by category. The docs page loads and displays them by path.

**Path convention:** `attributes/<category>/<filename>.cs`

**Categories** (match docs sidebar):

- `buttons/` — Button, EnumToggleButtons
- `collections/` — ListDrawerSettings, TableList
- `conditionals/` — DisableIf, EnableIf, HideIf, ShowIf, etc.
- `debug/` — ShowDrawerChain
- `decorators/` — DisplayAsString, Dropdown, InlineEditor, etc.
- `groups/` — BoxGroup, FoldoutGroup, HorizontalGroup, TabGroup, etc.
- `misc/` — EnumPaging, OnValueChanged, PropertyOrder, etc.
- `styling/` — GUIColor, HideLabel, Indent, LabelText, etc.
- `validators/` — AssetsOnly, InfoBox, Required, etc.

In `data/docs.ts`, set `usageSection.codeFilePath` to the path under `public/`, e.g. `attributes/buttons/Buttons_ButtonSample.cs`.
